# MyLifeEasy

**A unified platform for students and working professionals relocating to new cities in India.**

---

## Platform Overview

### The Problem

Moving to a new city in India is overwhelming. Students, interns, fresh graduates, and working professionals face fragmented solutions across multiple apps and websites:

- Finding accommodation requires browsing 5+ apps with unreliable listings
- Essential services (food, laundry, repairs) are scattered across platforms with no quality guarantee
- Career support is disconnected from daily life needs
- No single source of truth for city-specific information
- Trust and safety concerns with unverified service providers

### The Solution

**MyLifeEasy** is a city-centric platform that consolidates everything a newcomer needs to thrive in a new city‚Äîfrom finding verified accommodation to daily essentials, career growth, and community engagement. Think of it as your **"City OS"** for seamless urban living.

### Target Users

- **Students** relocating for college or university
- **Interns & Fresh Graduates** starting their first job
- **Working Professionals** transferring to a new city
- **Families** moving for career opportunities

- **Large Student Population**: Home to prestigious universities like IIT Madras, Anna University, and SRM

## User Flow

### Step 1: Onboarding
1. User signs up and selects their role (Student / Professional / Family)
2. Provides city, budget, and preferences
3. AI chatbot conducts a quick needs assessment

### Step 2: Finding Accommodation
1. Browse verified PG listings, hostels, or rental options
2. Filter by budget, location, amenities, and ratings
3. Book visits or directly confirm bookings
4. Connect with potential roommates through matching algorithm

### Step 3: Daily Services Setup
1. Subscribe to food/tiffin services based on dietary preferences
2. Set up water delivery and laundry schedules
3. Add local stores and repair contacts to personalized directory

### Step 4: Career & Growth
1. Explore internship and job opportunities in the city
2. Enroll in skill development courses
3. Access career coaching and resume building tools

### Step 5: Community Engagement
1. Join city events, meetups, and interest groups
2. Participate in book exchanges and knowledge sharing
3. Unlock exclusive vouchers and partner discounts

### Ongoing: AI Assistance
- 24/7 chatbot for instant help
- Weekly budget insights and spending tips
- Personalized recommendations for services and opportunities
- Safety alerts and city updates

## Getting Started

### Prerequisites
- Node.js (v18+)
- npm or yarn
- MongoDB instance (local or cloud)

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/MyLifeEasy.git

# Navigate to client directory
cd MyLifeEasy/Client

# Install dependencies
npm install

# Start development server
npm run dev
```

### Environment Variables

Create a `.env` file in the root directory:

```env
VITE_API_URL=http://localhost:5000
VITE_OPENAI_API_KEY=your_openai_key
VITE_FIREBASE_CONFIG=your_firebase_config
```
